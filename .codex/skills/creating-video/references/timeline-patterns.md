# Timeline Patterns

Timeline file: `~/GitHub/adithyan-ai-videos/src/data/timeline.json`

## Common Overlay Types

- `intro_title`
- `subtitle`
- `label` (RAW RECORDING)
- `codex_callout`
- `status_left`
- `box`, `box_track`, `box_move`
- `canvas`
- `tool_flow`
- `offline_notice`
- `pixel_pulse`
- `disclaimer`
- `outro`

## Typical Flow

1. Intro title
2. Subtitle
3. RAW RECORDING label
4. CODEX badge
5. Active speaker detection overlays
6. Canvas phase (sketch frame)
7. Tool flow list
8. Disclaimer
9. Outro

## Timing Rules of Thumb

- Keep each overlay readable for at least 3–5 seconds.
- Stagger list items by 1–3 seconds for clarity.
- Use a 10–15s window for disclaimers.
